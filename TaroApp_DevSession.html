01 · PROJECT OVERVIEW
🤖
CLAUDE

TaroCardDeckApp — A Qt6 C++ Android application featuring a Hindu Mythological Tarot deck. Full 78-card deck with each card mapped to a Hindu deity, archetype, mantra, and mythological notes. Dark temple-chic UI with saffron/gold accents.

Tech stack: Qt 6.11.0 · Android NDK r27d · ARM64-v8a · CMake + Ninja · QML frontend · C++ backend

App concept: Sacred divination tool combining Western tarot with Hindu mythology. Each card has an English name + Sanskrit name, Hindu deity mapping, upright/reversed keywords, core mantra, and archetype notes.

04 · CMAKE & ANDROID MANIFEST
CMakeLists.txt — FINAL
cmake_minimum_required(VERSION 3.21)
project(TaroCardDeckApp VERSION 1.0 LANGUAGES CXX)

set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)
set(CMAKE_AUTOMOC ON)
set(CMAKE_AUTORCC ON)
set(CMAKE_AUTOUIC ON)

find_package(Qt6 REQUIRED COMPONENTS Core Gui Widgets)
qt_standard_project_setup()

# Android SDK version — Qt 6.11 requires API 36+
set(QT_ANDROID_TARGET_SDK_VERSION 36)
set(QT_ANDROID_COMPILE_SDK_VERSION 36)

qt_add_executable(TaroCardDeckApp main.cpp)

target_link_libraries(TaroCardDeckApp PRIVATE
    Qt6::Core Qt6::Gui Qt6::Widgets)

set_target_properties(TaroCardDeckApp PROPERTIES
    QT_ANDROID_PACKAGE_SOURCE_DIR "${CMAKE_CURRENT_SOURCE_DIR}/android")

include/Card.h
#pragma once
#include <QString>
#include <QStringList>

struct Card {
    QString id;
    QString name;
    QString sanskritName;
    QString arcana;       // "Major" | "Cups" | "Swords" | "Pentacles"
    QString suit;
    QStringList uprightKeywords;
    QStringList reversedKeywords;
    QString hinduBeing;
    QString type;         // Deva | Devi | Rishi | Avatar | Asura | Human
    QString systemRole;
    QString mantra;
    QString notes;
    QString imagePath;
    bool isReversed = false;
};

09 · UI DESIGN SPECIFICATION
👤
CJ — UI VISION

Theme: Modern dark — deep pitch-black/velvet base, saffron-orange and gold accents. "Temple-chic" — sleek UI + temple-mythic icons.

Colors: #000000 background · #FFD700 gold · #FF4500 saffron highlights

Cards: Horizontally laid out, user scrolls horizontally, taps corner dot to select card into tray

Splash: Full-screen golden Om, pulsing saffron glow, golden particles, temple bell sound

Selection tray: Snake-border decoration, slides in from top

Fonts: Cinzel (headings) · Noto Sans Devanagari (Sanskrit) · Crimson Text (body)

ScreenDescription

DivinationHorizontal scrollable deck (78 cards), snake-border tray, drag-scroll, corner-dot select
ReadingSpread layout (1–3 cards), Past/Present/Future labels, reversed cards, mantra per position
Card DetailFull deity info, upright/reversed keywords, Hindu archetype block, core mantra, notes
CatalogueAll 78 cards searchable, filter by suit, tap to open detail
